## OrangeMsg

消息中心接口文档包括 **消息发送** 部分和 **消息接收** 部分，消息发送部分主要针对后端业务触发消息，如变更课程提醒；消息接收部分主要针对前端数据显示，如个人消息盒子等功能，主要提供了消息分页查看及查看消息详情等功能性接口。
